## Overview

This is a Jekyll-based personal blog hosted on GitHub Pages at `blog.imekaku.com`. It uses standard Jekyll with no plugins (aside from a vendored TOC include), no Gemfile, and no build toolchain beyond Jekyll itself.

## Common Commands

- **Build the site**: `jekyll build`
- **Serve locally**: `jekyll serve` (typically available at `http://localhost:4000`)
- **Clean build artifacts**: `jekyll clean`

## Architecture

### Content Types

The site has three distinct content types with different Jekyll constructs:

1. **Blog Posts** (`_posts/`): Standard Jekyll posts. Use `layout: post` and require a `category` frontmatter field. Rendered with `_layouts/post.html`.
2. **Life Collection** (`_life_collection/`): A Jekyll collection (defined in `_config.yml`) for travel and reading notes. Use `layout: post_life` and require a `category` frontmatter field. Rendered with `_layouts/post_life.html`.
3. **Thinking Collection** (`_thinking_collection/`): A Jekyll collection for short thoughts. Rendered as a single chronological list at `/thinking/` with `layout: thinking`.

### Category Pages

Categories are **not** auto-generated. Each category requires a static HTML file:
- Blog post categories: create a file in `category/` (e.g., `category/algorithm.html`) with `layout: category` and `category: algorithm` frontmatter.
- Life collection categories: create a file in `category_life/` (e.g., `category_life/travel.html`) with `layout: category_life` and `category: travel` frontmatter.

The category index pages (`category/index.html` and `category_life/index.html`) loop over `site.categories` and link to these static pages.

### Layouts

- `_layouts/default.html` — Base layout. Navigation tabs: blog (home), thinking, category, life, about.
- `_layouts/post.html` — Blog posts. Includes a TOC (`{% include toc.html html=content %}`) and a "Related Posts" section.
- `_layouts/post_life.html` — Life collection posts. Includes a TOC but no related posts.
- `_layouts/thinking.html` — Minimal wrapper for thinking items.
- `_layouts/category.html` / `_layouts/category_life.html` — Category listing pages.

### Table of Contents

TOC is generated by `_includes/toc.html`, a vendored copy of [allejo/jekyll-toc](https://github.com/allejo/jekyll-toc) v1.2.0. It is included in post layouts with `{% include toc.html html=content %}`.

### MathJax

Posts that need MathJax should include `formula` in their `tags` array. The `default.html` layout conditionally loads MathJax from a CDN only for those pages.

### Assets

- Stylesheets are in `css/` (`screen.css`, `syntax.css`).
- Images are hosted externally on `blogcdn.qihope.com`, not in this repo.

## File Conventions

- **Post filenames**: `YYYY-MM-DD-title.md`
- **Life collection filenames**: `YYYY-MM-DD-title.md` (or `YYYY-MM-DD-ISBN.md` for book notes)
- Frontmatter for posts must include `layout`, `title`, and `category`.
